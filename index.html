Termo de Aceite do Aplicativo Toque Toque

Última atualização:

Introdução

Bem-vindo ao aplicativo Toque Toque. Este documento estabelece os termos e condições para o uso do aplicativo, que oferece funcionalidades de segurança e conveniência para residências, como videochamadas, abertura de portas e portões, visualização de câmeras, recebimento de encomendas, acionamento de sirenes e controle de iluminação. Ao utilizar o aplicativo, você concorda com os termos aqui descritos.

Coleta de Dados Pessoais

Ao se cadastrar no aplicativo Toque Toque, solicitamos os seguintes dados pessoais:

  • Nome completo
  • Endereço
  • CPF
  • E-mail
  • Telefone

Uso dos Dados Pessoais

Os dados coletados serão utilizados para:

  • Autenticação e identificação do usuário
  • Gerenciamento e facilitação das funcionalidades do aplicativo
  • Comunicação com o usuário para envio de notificações e atualizações
  • Melhoria contínua dos serviços prestados

Compartilhamento de Dados

O Toque Toque não compartilha seus dados pessoais com terceiros, exceto quando necessário para cumprir obrigações legais ou quando expressamente autorizado por você.

Armazenamento e Proteção de Dados

Seus dados pessoais são armazenados de forma segura, utilizando medidas técnicas e administrativas adequadas para proteger contra acessos não autorizados, perda ou destruição.

Direitos dos Usuários

De acordo com a LGPD, você possui os seguintes direitos sobre seus dados pessoais:

  • Confirmação da existência de tratamento de dados
  • Acesso aos dados
  • Correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados desnecessários ou excessivos
  • Portabilidade dos dados a outro fornecedor de serviço ou produto
  • Eliminação dos dados pessoais tratados com consentimento
  • Informação sobre compartilhamento de dados
  • Revogação do consentimento

Para exercer esses direitos, entre em contato conosco através do e-mail contato@toquetoque.app.

Alterações no Termo de Aceite

O Toque Toque se reserva o direito de alterar este termo a qualquer momento. Notificaremos você sobre quaisquer alterações significativas através do aplicativo ou por e-mail. Recomendamos que revise periodicamente este documento para estar ciente de quaisquer modificações.

Contato

Se você tiver dúvidas sobre este termo de aceite ou sobre o tratamento de seus dados pessoais, entre em contato conosco pelo e-mail contato@toquetoque.app ou pelo telefone [telefone de contato].

Aceitação dos Termos

Ao clicar em "Aceito", você declara que leu, compreendeu e concorda com todos os termos e condições estabelecidos neste documento.
